Why clauptain-hook?

See what changes when you stop writing hooks by hand.

Type-safe everything

Raw hooks give you unstructured JSON piped through stdin. Typo in a field name? Silent bug. Clauptain-hook gives you real TypeScript types for every event — mistakes are caught at compile time, not production.

Smart type narrowing

Pass a tool name to the matcher and the callback receives the exact input type for that tool. "Write" gives you file_path and content. "Bash" gives you command. No manual narrowing.
